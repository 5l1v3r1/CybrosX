Assignments: list

Lists the assignments by task.

Request

GET /v1/assignments/

Request Parameters

Field Name Data Type Description
task_id integer Task ID to fetch the assignments for.

Response

{{ '{' }}

]

count integer The number of assignments that were submitted for this project.

previous string URL to fetch previous page.
results [ ] list The list of assignments submitted by workers.
